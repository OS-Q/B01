### [EDGE-Q描述](https://github.com/OS-Q/EDGE-Q/wiki) 

EDGE-Q是边缘设备总称，通过大量的边缘节点设备实现功能的组合叠加，使任意的系统结构成为可能。

[EDGE-Q](https://github.com/OS-Q/EDGE-Q) 同样也是各种解决方案的总称，通过不同解决方案模块实现实现更强大的功能组合。
